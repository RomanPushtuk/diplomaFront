## Запуск проект

### `npm start`

Запуск проекта в режиме разработки<br />
откроется на порту http://localhost:3000 в браузере

### `npm test`

Запуск тестов<br />

### `npm run build`

Собрать проект<br />

### `npm run eject`

Разобрать проект (станут доступны файлы сборщика, можно сделать только один раз)<br />
